## 🔭 Featured Project: CafBot

[![Live Demo](https://img.shields.io/badge/Live_Demo-https://chatbot--180426.onrender.com-46E3B7?style=for-the-badge&logo=render)](https://chatbot-180426.onrender.com)
[![GitHub Repository](https://img.shields.io/badge/GitHub-View_Code-181717?style=for-the-badge&logo=github)](https://github.com/lozanoMiguel/chatbot)

**An AI-powered coffee assistant deployed in production**

CafBot helps customers choose the perfect coffee based on brewing method (espresso/filter) and flavor profile (traditional/exotic/funky). Built with a **hybrid architecture** that combines deterministic Python logic (for reliability) with LLMs (for natural conversation).

### ✨ Key Features

| Feature | Implementation |
|---------|----------------|
| **Hallucination-free recommendations** | Core logic in Python, LLMs only for descriptions |
| **RAG for accurate information** | ChromaDB vector search over coffee knowledge base |
| **Persistent memory** | Conversation history in PostgreSQL (Supabase) |
| **Containerized** | Docker image runs identically in dev and production |
| **Cloud deployment** | Live on Render with CI/CD from GitHub |
